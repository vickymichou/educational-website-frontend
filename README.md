# 🎓 Educational Website (Frontend)

A multi-page educational website built using HTML, CSS, and JavaScript, designed to simulate a simple course platform interface.

## 📌 Overview

This project presents a structured educational website where users can navigate through announcements, documents, homework, and communication pages. It focuses on frontend development and user interface design.

## 🚀 Features

- Multi-page navigation (Home, Announcements, Documents, Homework, Communication)
- Clean and simple user interface
- Structured layout for educational content
- Downloadable resources and materials
- Contact page for user interaction

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript


## 🎯 Purpose

This project was developed to practice frontend development concepts, including:

- Page structuring
- Navigation design
- Content organization
- Basic UI/UX principles

## ⚠️ Notes

This is a **frontend-only project** with no backend or database integration.
